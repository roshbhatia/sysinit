## Install

```bash
# Install dependencies
./install-deps.sh

# Resolve perms issues.
sudo chown -R $(whoami):staff /nix
sudo chown -R $(whoami):staff $HOME

# Build and activate
task refresh

# Update all flake inputs
task update-flake

# Cleanup old generations
task clean
```
